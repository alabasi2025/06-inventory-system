# ملخص القواعد الصارمة لنظام المخازن

## القواعد الأساسية للتشغيل:

### 1. البنية التقنية
- **Nx Monorepo**: إلزامي لإدارة المشروع
- **NestJS للـ Backend**: إلزامي (المنفذ 3006)
- **Angular للـ Frontend**: إلزامي (المنفذ 4206)
- **PostgreSQL**: قاعدة البيانات الإلزامية
- **Prisma**: إلزامي كـ ORM
- **PrimeNG + TailwindCSS**: للواجهات

### 2. خطوات التشغيل الإلزامية
1. تثبيت PostgreSQL
2. إنشاء قاعدة البيانات والمستخدم
3. تشغيل `pnpm install`
4. تشغيل `npx prisma generate`
5. تشغيل `npx prisma db push`
6. تشغيل API: `npx nx serve api`
7. تشغيل Web: `npx nx serve web`

### 3. المنافذ
- API: 3006
- Web: 4206

### 4. قواعد مهمة
- لا شاشات بدون Backend
- لا بيانات وهمية
- CRUD كامل لكل شاشة
- معالجة الأخطاء إلزامية
